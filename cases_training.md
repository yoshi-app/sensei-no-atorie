## 5. 振り返り（4事例）

### 5-1. 3点振り返りシート
| フィールド | 内容 |
|---|---|
| 対象者レベル | 初心者 |
| 活動タイプ | 振り返り |
| 所要時間 | 15分 |
| 活動名 | 3点振り返りシート |
| 概要 | 「気づいたこと」「疑問に思ったこと」「明日からやること」の3点を個人で記入し、隣の人と1分シェアする。 |
| 人数規模 | 個人→ペア |
| 産出物 | 振り返りシート |
| 現場適用性 | 高 |
| ファシリ負荷 | 低 |
| 必要なもの | 振り返りシート（事前準備） |
| 評価の方法 | 成果物確認 |
| 設計のポイント | 「明日からやること」は動詞で書かせる（「調べる」ではなく「〇〇をChatGPTに入力してみる」など具体的に）。 |

### 5-2. アクションプラン作成
| フィールド | 内容 |
|---|---|
| 対象者レベル | 中級 |
| 活動タイプ | 振り返り |
| 所要時間 | 30分 |
| 活動名 | アクションプラン作成 |
| 概要 | 研修で得た学びをもとに、業務での具体的な活用計画（What・When・How）を1枚にまとめ、ペアでレビューし合う。 |
| 人数規模 | 個人→ペア |
| 産出物 | アクションプランシート |
| 現場適用性 | 高 |
| ファシリ負荷 | 低 |
| 必要なもの | アクションプランシート（事前準備） |
| 評価の方法 | 成果物確認・ペアフィードバック |
| 設計のポイント | 「いつまでに」の期限を必ず書かせる。2〜4週間後にフォローアップの場を設けると実践率が上がる。 |

### 5-3. 学びの一言シェア
| フィールド | 内容 |
|---|---|
| 対象者レベル | 初心者 |
| 活動タイプ | 振り返り |
| 所要時間 | 15分 |
| 活動名 | 学びの一言シェア |
| 概要 | 各自が「本日の最大の気づき」を一言で発表し、全体で共有する。研修のまとめとして全員の声を可視化する。 |
| 人数規模 | 全体 |
| 産出物 | 付箋（またはデジタルボード）への投稿 |
| 現場適用性 | 中 |
| ファシリ負荷 | 低 |
| 必要なもの | 付箋・ホワイトボード（またはMentimeter等） |
| 評価の方法 | 観察 |
| 設計のポイント | ファシリテーターが類似の気づきをその場でグルーピングして見せると、学びの全体像が見えやすくなる。 |

### 5-4. 実務適用宣言
| フィールド | 内容 |
|---|---|
| 対象者レベル | 中級 |
| 活動タイプ | 振り返り |
| 所要時間 | 15分 |
| 活動名 | 実務適用宣言 |
| 概要 | 「来週中に〇〇でAIを使う」と具体的に宣言し、ペアで互いに証人になる。コミットメントを言語化・社会化する。 |
| 人数規模 | ペア |
| 産出物 | 宣言カード |
| 現場適用性 | 高 |
| ファシリ負荷 | 低 |
| 必要なもの | 宣言カード（名刺サイズ・事前印刷） |
| 評価の方法 | 成果物（宣言内容の具体性） |
| 設計のポイント | 宣言は「業務名＋使い方＋期限」の3点セットで書かせる。カードは持ち帰らせてデスクに貼るよう促す。 |

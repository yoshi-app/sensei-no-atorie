# 学びデザインシステム

## 概要

体験的・ゲーム的学習の事例研究と要素分解を基に、授業・講座の設計を支援するシステム。

11の観点（活動の形、身体性、使うもの、問いの性質、他者との関係、産出物、現実との接点、時間軸、評価の対象、評価の方法、評価のタイミング）で、48の学習活動事例を分類・整理。

最終的には **AI講座ビルダー** と融合し、ワークを含む設計書・原稿レベルでの講座自動生成を実現する。

---

## 構成

### リサーチ資料
- **research_results.md** — 国際事例、ツール、フレームワーク、効果分析を含む包括的なリサーチ
- **cases_analyzed.md** — 48事例の要素分解（11観点×7カテゴリ）

### ファイル構成

```
学びデザインシステム/
├── README.md（このファイル）
├── セッションノート.md
├── research_results.md（リサーチ結果）
├── cases_analyzed.md（要素分解結果）
└── 構想・設計/（今後追加予定）
    ├── システム設計書
    ├── ワークフロー
    └── AI講座ビルダー連携仕様
```

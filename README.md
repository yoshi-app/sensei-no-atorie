# 学びデザインシステム

## 概要

体験的・ゲーム的学習の事例研究と要素分解を基に、授業・講座の設計を支援するシステム。

11の観点（活動の形、身体性、使うもの、問いの性質、他者との関係、産出物、現実との接点、時間軸、評価の対象、評価の方法、評価のタイミング）で、48の学習活動事例を分類・整理。

最終的には **AI講座ビルダー** と融合し、ワークを含む設計書・原稿レベルでの講座自動生成を実現する。

### 使用方法

1. **ブラウザで開く**
   ```bash
   # ローカル環境で開く場合
   open index.html

   # または、簡易サーバを起動（Python 3.x）
   python -m http.server 8000
   # ブラウザで http://localhost:8000 にアクセス
   ```

2. **条件を入力して検索**
   - 教科、学年、活動の形から条件を選択
   - 「検索」ボタンをクリック
   - マッチした事例がカード表示される

3. **詳細を確認して採用**
   - 事例カードをクリックして詳細モーダルを表示
   - すべての11観点を確認
   - 「この事例を採用」ボタンで設計メモテンプレートをコピー
